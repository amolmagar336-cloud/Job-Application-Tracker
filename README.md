# Job Application Tracker

A full-stack web application to track job applications and interview status.

## Features

- Add new job applications
- Update application status
- Delete applications
- Dashboard analytics with charts
- Search and filter jobs

## Tech Stack

Frontend
- HTML
- CSS
- JavaScript

Backend
- Node.js
- Express

Database
- SQLite

Charts
- Chart.js

## Installation

```bash
npm install
node backend/server.js
